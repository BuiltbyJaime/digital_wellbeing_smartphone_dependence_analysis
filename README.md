# Keystone Project: Screen Time and Digital Addiction Analysis

This project explores patterns in screen time habits and their relationship with stress, sleep, and digital addiction.

The dataset contains information about users' daily screen time, social media usage, gaming habits, sleep duration, and addiction levels.

The goal is to clean and explore the data to identify trends and relationships between technology usage and wellbeing.

The requirements for the project will be listed in the requirements.txt file.

The three main questions that will be observerd are:

    1. Does higher screen time relate to higher addiction levels?

    2. How does sleep duration differ between addicted and non-addicted users?

    3. Is stress level associated with screen addiction?
